# Business Intelligence Dashboard

## Overview
This project focuses on analyzing business performance using Power BI to support data-driven decision-making.

## Tools Used
- Power BI
- Power Query
- Microsoft Excel

## Project Objectives
- Analyze business performance.
- Create interactive dashboards.
- Track key performance indicators (KPIs).
- Present insights through data visualization.

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- Data Visualization
- Dashboard Development
- Business Intelligence
